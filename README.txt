Запуск:  
./bin/pcscale

Для указания файла
./bin/pcscale -i весовуха.csv -e cp1251


Usage: pcscale [-e=<encode>] [-i=<inputFile>] [-o=<outputFile>]
convert lerua.csv to pc200 format
  -e, --encode=<encode>     encoding for lerua.csv and pc200
  -i, --input=<inputFile>   lerua.csv input file
  -o, --output=<outputFile> output pc200

Значения по умолчанию: 
--encode=cp1251
--input="весовуха.csv"
--output="PCScale.txt"
